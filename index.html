Project Description

GAA Connect is a Progressive Web App designed to streamline GAA club management by centralising fixtures, results, player statistics, and communication. It allows simple interaction between coaches, managers, players, and supporters and provides role-based access for individual users. As a web-based platform, it provides offline match data tracking capabilities and ensures easy access across devices. GAA Connect modernises club operations, enhancing game preparation, team management, and growth for players.
